\# Lung Sound Classification for Respiratory Disease Using CNN-LSTM



\## Overview



This project focuses on the classification of respiratory diseases using lung sound recordings and a CNN-LSTM deep learning model. The system processes lung sound audio files, extracts features, and classifies respiratory conditions based on learned acoustic patterns.



\## Objectives



\* Analyze lung sound recordings for disease detection.

\* Extract meaningful audio features from respiratory sounds.

\* Train a CNN-LSTM model for accurate classification.

\* Assist in early detection of respiratory diseases using AI.



\## Technologies Used



\* Python

\* PyTorch

\* NumPy

\* Pandas

\* Librosa

\* CNN-LSTM Deep Learning Model



\## Dataset



The project uses respiratory sound recordings along with patient diagnosis information. Audio files are preprocessed and transformed into feature representations suitable for deep learning.

\## Workflow



1\. Collect respiratory sound recordings.

2\. Preprocess audio files.

3\. Extract audio features.

4\. Train CNN-LSTM model.

5\. Evaluate model performance.

6\. Predict respiratory disease classes.



\## Features



\* Lung sound preprocessing

\* Feature extraction

\* Deep learning-based classification

\* Model training and evaluation

\## Results



The CNN-LSTM model successfully learns temporal and spatial patterns from respiratory sounds, enabling effective classification of respiratory diseases.
